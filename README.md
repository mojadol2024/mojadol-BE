# MOJADOL
안드로이드 앱
## 🖥️ 프로젝트 소개
잃어버린 강아지의 견종을 알아내고 제보 발생 시 유사한 강아지를 찾아 알림을 주는 안드로이드 애견 어플입니다.

## ⚙️ 개발환경 
- `java 17`
- `jwt 0.11.2`
- `spring security 3.1.3`
- `lombok`
- **IDE** : STS 4.0
- **Framwork** : spring boot 3.1.3
- **Database** : mysql 8.0, redis
- **ORM** : JPA
- **CI/CD** : dockerHub, GitHub Actions
- **API** : Kakao Developers, firebase
